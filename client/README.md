# 这是什么

一个管理后台(不带服务端，只是一个 demo,数据是 mock),目的是为了学习 React+antd

# 主要功能

用户管理

角色管理

菜单管理

权限管理

# 技术栈

react +antd+tailwindcss

状态管理器，自己实现的简化版本 zustand

路由，自已实现的简化版本的 react-router

# 使用方法

```
npm install

npm run dev
```

浏览器访问 127.0.0.1:对应端口即可
